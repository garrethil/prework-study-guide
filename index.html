Prework Study Guide

HTML

  • HTML organizes the framework of a webpage and it's contents
  • The head portion of HTML holds meta data that helps the browser render the page but is mostly not visible to the user
  • The body portion of HTML contains all the information and images that are visible to the user. there are parent elements and semantic or child elements

CSS

  • A margin indicates how much space we want outside of an element
  • A padding indicates how much room we want around content inside an element

Git

  • git status: checks what branch we are currently on
  • git checkout -b Branch-Name: creates a new branch and moves you to it
